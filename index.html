Import de candidats depuis Google Sheets (CSV)
Format CSV Google Sheets détecté :

Votre formulaire contient 15 colonnes qui seront automatiquement mappées :

  1. Horodateur → Date de début
  2. Nom d'utilisateur → Ignoré
  3. Nom & Prénom → Séparé automatiquement
  4. Email → Email du candidat
  5. Téléphone → Numéro de contact
  6. Ville + mobilité → Localisation
  7. Type de contrat → CDI/CDD/Stage/Alternance/Freelance
  8. Stack & expérience → Objectif professionnel enrichi
  9. Lien CV/LinkedIn → Portfolio
Traitement automatique :
  • ✅ Séparation nom/prénom automatique
  • ✅ Normalisation téléphone (format français)
  • ✅ Type de contrat reconnu intelligemment
  • ✅ Objectif enrichi (stack + niveau + souhaits)
  • ✅ Date d'horodateur → date de début
  • ✅ Déduplication par email
  • ✅ Validation complète des données

Export depuis Google Sheets :

  1. Fichier → Télécharger → CSV
  2. Cliquer "Importer" ci-dessus
  3. Choisir : ajouter ou remplacer
